API Sateré-Mawé

Esta é a documentação de referência para a API sateré-mawé. A própria API é baseada em recursos representados pelo formato JSON e manipulados usando o protocolo HTTP..


Versões da API: Todas as alterações importantes nas APIs sataré-mawé serão documentadas aqui. Atualmente, as APIs sateré-mawé estão na versão v1. Você pode fazer uma requisição em python da seguinte forma para ter acesso em formato JSON. Exemplo: 

            import   requests
            import  json
            url='http://127.0.0.1:5000/api/numeros_satere' 
            response = requests.get(url) 
           if response.status_code == 200: 
              data = response.json()  
              for numero_satere in data: 
                    print(f"ID: {numero_satere['id']}, Número: {numero_satere['numero']}, Descrição: {numero_satere['descricao']}")
          else:
              print( "Falha na requisição. Verifique se a API está em execução.")

Antes de executar este código, é importante garantir que o seguinte esteja configurado:

  • Python: Certifique-se de que você tenha o Python instalado em seu sistema.
  • Biblioteca Requests: Você deve ter a biblioteca 'requests' instalada para realizar solicitações HTTP. Você pode instalá-la com o seguinte comando:

              pip install requests

O código em questão realiza as seguintes operações: Importa as bibliotecas necessárias:

  • requests: Esta biblioteca é usada para fazer solicitações HTTP.
  • json: Esta biblioteca é usada para analisar os dados JSON recebidos como resposta da API.

Métodos HTTP são usados ​​para manipular recursos. Porém, como nem todos os recursos permitem todas as operações HTTP, observe OK a referência de cada recurso abaixo. Métodos usados ​​com endpoints de coleta, no momento está api só aceita apenas requisições get. observe:

Visualizar na WEB: Para fazer uma requisição a está API externa e exibir os dados no navegador usando Flask, você pode criar uma rota em sua aplicação Flask que faz a requisição à API externa e depois exibe os dados na página HTML. Aqui está um exemplo de como fazer isso: 

       from flask import Flask, render_template  
          import requests 
          app = Flask(__name__)
          @app.route('/')
          def show_data():
              url ='http://127.0.0.1:5000/api/numeros_satere' 
              response = requests.get(url)

              if response.status_code == 200:
                  data = response.json()
                  return render_template ('data.html', numeros_satere=data)
              else:
                  return "Falha na requisição. Verifique se a API está em execução." 

          if __name__ == '__main__':
              app.run(debug=True)

Neste exemplo, criamos uma rota raiz ('/') que faz a requisição à API externa e, se a resposta for bem-sucedida, renderiza um modelo HTML para exibir os dados. Se a resposta da API não for bem-sucedida, uma mensagem de erro será exibida. Você também precisará criar um arquivo HTML chamado data.html para exibir os dados.
